모든 html 요소에는 두 가지의 attribute(속성값)를 부여해 줄 수 있다.

ex) id ==> 유일 ///
    <a id="my">dfd></a> 이런 id의 호출은 #my #붙힌다 

    class ==> 겹쳐서 가질 수도, 여러개 가질 수도 있음 / 클래스는 .으로 호출
    <a class="b c d"
    <div class="b"

<a href="#" targe="">속성</a>

target
      _self ==> 현재 페이지에서 이동(기본값 self)
      _blank ==> 현재 페이지를 냅두고 새로운 창으로 이동
      _parent ==> 부모 프레임이 있다면 부모 페이지에서 이동
      _top ==> 최상위 페이지 이동

<a 요소는 기본적으로 두 가지 역할을 한다. >
1. 링크로써의 기능 : 속성으로 href를 쓰며 값으로 해당주소 또는 #이름을 사용한다.
2. 링크의 타깃이 되는 기능 : name 속성을 가지는 것이나, 지금은 id 속성으로 대체 되었다.
                                    id속성은 모든 요소가 가질 수 있는 속성이기 때문에 굳이 타깃을 위해 
                                    a 요소를 사용할 필요가 없다.
                                    a 요소 안에 텍스트는 접근성을 위해, 앞 뒤 문맥 없이도 해당 링크 페이지를 
                                    알기 쉽게 넣어야 한다.

<a href ="#"> ==> 페이지 제일 위로 보내게 해줌

ul(점으로 찍히는 리스트) 자식으로 li 태그를 반드시
ol(숫자로 찍히는 리스트) 자식으로 li 태그를 반드시
dl > 자식으로 dt(개념), dd(정의) 두가지 태그를 가짐, 1대다 성립
<dl>
    <dt>이름</dt>
    <dd>홍길동</dd>   
</dl>

ol: 순서가 있는 목록
ul: 순서가 없는 목록

ol,ul 요소는 자식으로 li 요소만 올 수 있다.
li 는 블럭 요소 이므로 대부분의 블럭 요소를 가질 수 있다.

dl은 개념 - 정의를 나타내는 목록
dl 요소는 자식으로 dt, dd를 가질 수 있다.
dt 는 개념,dd는 정의를 나타낸다.
dt, dd 는 1:1, 1:다, 다:1, 다:다 형식 모두 쓸 수 있다.
