var 는 모든 함수 스코프의 적용을 받음
let 은 모든 스코프의 적용을 받음

함수 밖에 쓴 var 변수1 과 함수 안에쓴 var 변수1은 다른 변수이다.
그러나 if문등 다른 스코프에서는 같은 변수이다.

모든 스코프 밖에 쓴 let 과 안에 쓴 let은 다른 변수이다.

var num = 1;  

if(1) {
   var num = 3;
}

num 출력시 3이 됨.
------------------------------
let num = 1;

if(1) {
   let num = 3;
}
let 출력시 1이 됨.
